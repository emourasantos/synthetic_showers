# synthetic_showers
The files chuveiros_bari_sig_xxstats.txt contains information on synthetic extensive air showers in the format:

column 01: N # of triggered detectors
column 02: sigi (uncertainty on trigger time) [ns]
column 03: xb (x-coordinate of shower barycenter at ground level) [m]
column 04: yb (y-coordinate of shower barycenter at ground level) [m]
column 05: theta_v (shower true incoming direction) [deg]
column 06: phi_v (shower true incoming direction) [deg]
column 07: x1 (1st triggered station x-coordinate) [m]
column 08: y1 (1st triggered station y-coordinate) [m]
column 09: t1 (1st triggered station trigger time) [ns]
...
column 6+N  : xN (Nth triggered station x-coordinate) [m]
column 6+N+1: yN (Nth triggered station y-coordinate) [m]
column 6+N+2: tN (Nth triggered station trigger time) [ns]
